# Valida Contratos PDF 

## Pré-requisitos:

Para Windows Users:
- Instalar o Chocolatey seguindo as instruções contidas no link [Chocolatey](https://chocolatey.org/install).

- Instalar Vagrant:
  ```cmd
  choco install vagrant
  ```

Para Linux Users:
- Instalar Ruby
- Instalar Cucumber

## Como executar os scripts:

vagrant up
